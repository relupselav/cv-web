# Portafolio Personal – Valentina Spuler

Aplicación web creada con **Next.js**, **React** y **TypeScript** para presentar mi perfil profesional, experiencia laboral, habilidades y proyectos destacados.  
El objetivo del proyecto es servir como CV interactivo y carta de presentación digital.

## Tecnologías

- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS Modules
- Arquitectura basada en componentes
- Diseño original en Figma

## Funcionalidades

- Secciones: sobre mí, experiencia, proyectos y contacto
- Diseño responsivo utilizando CSS sin frameworks
- Optimización SEO básica
- Navegación con App Router (layout, páginas y segmentos)
- Carga rápida gracias al soporte SSR/ISR de Next.js
